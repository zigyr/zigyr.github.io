
在`_config.yml`中

添加`math: katex`

```yml
kramdown:
  footnote_backlink: "&#8617;&#xfe0e;"
  syntax_highlighter: rouge
  syntax_highlighter_opts: # Rouge Options › https://github.com/jneen/rouge#full-options
    css_class: highlight
    # default_lang: console
    span:
      line_numbers: false
    block:
      line_numbers: true
      start_line: 1
  math: katex
```


随后在每篇需要渲染的笔记的front matter处

添加`math: true`
