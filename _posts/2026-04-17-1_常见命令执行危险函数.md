## 分类总览
---

| 分类  | 函数                           | 回显能力                | 返回值形式              |
| --- | ---------------------------- | ------------------- | ------------------ |
| 无回显 | `` ` ``、`shell_exec`、`popen` | 需要手动 `echo`         | 字符串                |
| 有回显 | `exec`、`passthru`、`system`   | 自动输出全部结果            | —                  |
| 特殊  | `pcntl_exec`、`proc_open`     | `pcntl_exec` 依赖程序设计 | `proc_open` 通过管道返回 |

## 无回显函数
---

### 反引号


```php
echo `$cmd`, PHP_EOL;
```

### shell_exec


```php
$output = shell_exec($cmd);
echo $output;
```

### popen

```php
$handle = popen($cmd, 'r');
while ($content = fgets($handle)) {
 print_r($content);
}
pclose($handle);
```

## 有回显函数
---

### exec

执行命令，完整结果存入数组，通过 `print_r` 输出。

直接 `echo exec($cmd)` 仅回显命令结果的**最后一行**。

```php
exec($cmd, $array);
print_r($array);

// 仅最后一行
echo exec($cmd);
```

### passthru


```php
passthru($cmd);
```

### system


```php
system($cmd);
```


## proc_open
---

> proc_open：通过描述符数组控制输入/输出/错误流的管道函数，支持灵活的重定向和流控制

### 描述符规范

- `0`：标准输入（stdin）
- `1`：标准输出（stdout）
- `2`：标准错误（stderr）

支持格式：`["pipe", "r"]`、`["pipe", "w"]`、`["file", "路径", "a"]`、`["null"]`

- `$pipes[0]` 输入流、`$pipes[1]` 输出流、`$pipes[2]` 错误流，索引严格对应
- `["null"]` 可禁用对应流，常用于绕过过滤

配合 `stream_get_contents()` 读取管道内容，`proc_close()` 关闭进程

### 标准输出回显

`?cmd=cat flag`

```php
$descriptors = [
 ["pipe", "r"],
 ["pipe", "w"],
 ["file", "/tmp/error-output.txt", "a"]
];
$process = proc_open($cmd, $descriptors, $pipes);
echo stream_get_contents($pipes[1]);
proc_close($process);
```

### 仅回显错误输出

当 stdout 被禁用时，正常输出不会显示，可通过将输出重定向到 stderr 来获取结果

`?cmd=cat flag >&2`


```php
$descriptors = [
 ["pipe", "r"],
 ["null"],
 ["pipe", "w"]
];
$process = proc_open($cmd, $descriptors, $pipes);
echo stream_get_contents($pipes[2]);
proc_close($process);
```

## pcntl_exec
---

非 PHP 核心内置函数，需要安装并开启 `pcntl` 扩展

仅支持指定可执行二进制文件路径，不支持直接执行命令语句。

```php
pcntl_exec("/bin/ls", ["-l"]);
```

## Q&A
---

### Q：什么是外部命令
A：

操作系统原生可执行程序或指令，非 PHP 代码：

- Linux：`ls`、`cat`、`whoami`
- Windows：`dir`、`ipconfig`
- 通用：`ping`、`python`、`node`

### Q：CLI （Command-Line Interface命令行界面）环境限制
A:

`pcntl_exec()`依赖 CLI 运行环境

当 PHP 脚本通过终端 `php 文件名.php` 启动时，该函数生效

以其他方式（如 Web 请求）运行时可能不生效或直接报错
