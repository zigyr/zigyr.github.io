## Linux文件读取替代命令
---

### 1. tac

反向倒序显示文件内容

`tac flag.php`

### 2. more

分页逐页显示文本内容，适用于文件内容过多、终端无法回滚的场景

但在flag文本读取中，效果类似`cat`

`more flag.php`

### 3. less

功能与 `more` 命令效果类似，同为文件分页查看工具。

`less flag.php`

### 4. tail

查看文件末尾指定行数内容，默认回显最后**10行**

文件内容较少时，可直接使用该命令读取全部内容

`tail flag.php`


### 5. nl

跟`cat`差不多，只是多了行号显示

`nl flag.php`

### 8. sort

默认用于文件内容排序，题目中被关键词过滤，可通过**特殊绕过写法**使用：
```shell
# 绕过写法
s\ort fl\ag.p\hp
/usr/bin/s?rt fl\ag.p\hp
```

常规使用示例
```shell
┌──(root㉿zigyr)-[~/command_lab]
└─# sort flag.php
2row: 66666
3row: hello
1row: there is a flag
```

### 9. uniq

用于报告或去除文件中重复的行，可正常读取文件内容
```shell
┌──(root㉿zigyr)-[~/command_lab]
└─# uniq flag.php 
1row: there is a flag
2row: 66666
3row: hello
```

### 10. file -f

以文件每行内容作为文件名进行读取操作，报错回显

```shell
┌──(root㉿zigyr)-[~/command_lab]
└─# file -f flag.php 
\3471row: there is a flag: cannot open `\3471row: there is a flag' (No such file or directory)
2row: 66666:               cannot open `2row: 66666' (No such file or directory)
3row: hello:               cannot open `3row: hello' (No such file or directory)
```

### 11. grep
在文件中检索、匹配指定字符串，快速提取包含目标内容的行
```shell
# 模糊匹配含fl的文件名中含有fla的内容
┌──(root㉿zigyr)-[~/command_lab]
└─# grep fla fl*
1row: there is a flag
```
