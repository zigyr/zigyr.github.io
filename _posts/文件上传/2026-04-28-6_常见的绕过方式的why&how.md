## 第六关 · 大写绕过

---
### 题目


```php
$deny_ext = array("...一堆黑名单");

$file_name = trim($_FILES['upload_file']['name']);
$file_name = deldot($file_name); // 删除文件名末尾的点
$file_ext = strrchr($file_name, '.'); // 以最后一个点作为分割，获取文件后缀
$file_ext = str_ireplace('::$DATA', '', $file_ext); // 去除字符串::$DATA
$file_ext = trim($file_ext); // 首尾去空

if (!in_array($file_ext, $deny_ext)) {
    $temp_file = $_FILES['upload_file']['tmp_name'];
    $img_path = UPLOAD_PATH . '/' . date("YmdHis") . rand(1000, 9999) . $file_ext;
    if (move_uploaded_file($temp_file, $img_path)) {
        $is_upload = true;
    } else {
        $msg = '上传出错';
    }
} else {
    $msg = '不允许上传.asp, .aspx, .php, .jsp后缀文件';
}
```

缺少`$file_ext = strtolower($file_ext);`导致的

### 前置知识点

在 Windows 中创建文件时，命名会把文件后缀自动转换——如 `.Txt` → `.txt`，`.Php` → `.php`

### HOW

1. 上传 `111.php` 的一句话木马
2. 抓包，修改文件名为 `111.pHP`
3. 在新的标签页打开图片：`basis_url/upload/时间戳随机数.pHP`
4. 复制 URL 至蚁剑，连接地址为 `basis_url/upload/时间戳随机数.pHP`

## 第九关 · $DATA 绕过
---

### 前置知识点

Windows，文件名后跟着 `::$DATA` 

表示文件的一个附加数据流（Alternate Data Stream, ADS）

是一种用于在文件内部存储额外数据的机制（如文件的元数据、备份信息、标签等）


普通情况下，文件只有一个默认的数据流，可以通过文件名访问

但 Windows NT 文件系统（NTFS）支持在文件内部创建额外的数据流，以存储其他信息

这些额外的数据流，可以通过在文件名后添加 `::$DATA` 来访问

> 需要注意的是，大多数常规的文件操作工具不会意识到这些额外的数据流，而只会处理默认的数据流。要访问或操作这些额外数据流，通常需要使用特定的命令行工具或编程接口。

### ADS 基本操作

- **准备**

```cmd
echo "1111" > 1.txt
echo "2222" > 2.txt
```

- **写入**

```cmd
echo "你好" >> 1.txt:nihao
type 2.txt >> 1.txt:this_2
```

- **读取**

```cmd
notepad 1.txt:nihao
notepad 1.txt:this_2
```


### HOW

1. 上传 `111.php`
2. BP 拦截后更改为 `111.php::$DATA`，放行上传
3. 上传成功后，在新标签页打开图片：

## 第十关 · 点空格点绕过
---

### 题目


```php
$deny_ext = array("...一堆黑名单");

$file_name = trim($_FILES['upload_file']['name']);
$file_name = deldot($file_name); // 删除文件名末尾的点
$file_ext = strrchr($file_name, '.'); // 以最后一个点作为分割，获取文件后缀
$file_ext = strtolower($file_ext); // 转小写
$file_ext = str_ireplace('::$DATA', '', $file_ext); // 去除字符串::$DATA
$file_ext = trim($file_ext); // 首尾去空

if (!in_array($file_ext, $deny_ext)) {
    $temp_file = $_FILES['upload_file']['tmp_name'];
    $img_path = UPLOAD_PATH . '/' . $file_name;
    if (move_uploaded_file($temp_file, $img_path)) {
        $is_upload = true;
    } else {
        $msg = '上传出错';
 }
} else {
    $msg = '不允许上传.asp, .aspx, .php, .jsp后缀文件';
}
```


### HOW

1. 上传 `111.php` 的一句话木马
2. 抓包，修改文件名为 `111.php. .`
3. 在新的标签页打开图片：`basis_url/upload/111.php.`
4. 复制 URL 至蚁剑，连接地址为 `basis_url/upload/111.php`


### WHY

 `deldot()` 
 
 从右向左**逐个删除末尾连续的点号**，直到遇到非点字符才停下

所以 `111.php. .` 经过 `deldot()` 后变为 `111.php. ` （末尾的点被删，空格保留），再经 `trim()` 后，提取到的后缀是 `. ` （点+空格），不在黑名单中

而 `$img_path` 拼接用的是原始 `$file_name`，即 `111.php. .`

Windows 会自动去除末尾的点和空格，最终文件名为 `111.php`
